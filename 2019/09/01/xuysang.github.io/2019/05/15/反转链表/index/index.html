反转链表（leetcode 206)

反转一个单链表。

示例:

输入: 1->2->3->4->5->NULL
输出: 5->4->3->2->1->NULL

解法：

一开始看到这个解法我是懵逼的，然后看解法下面的评论，开始了解这个表达方式，其实包含了一种我不知道的意思。就是while循环中的模块，是一个元祖形式，可以写为（rev,rev.next,p)=(p,rev,p.next)（这里可以看出，有看不懂的表达式是很正常的，这时候就要多搜索，从而获得更多知识点，逗号放在print中，可以不换行，具体的用法可以参考python多元赋值 https://blog.csdn.net/qq277511467/article/details/50915739

之后的理解相对就好些了。看到回复中有个便于理解的回复，下面贴过来：
前置条件：迭代指针：p = head、结果指针：rev = none
以1->2->3->4->5为例：
过程：
rev:None
第一层循环
rev:1->2->3->4->5 rev = p
rev:1->None rev.next = rev
p:2->3->4->5 p = p.next
第二层循环
rev:2->3->4->5 rev = p
rev:2->1->None rev.next = rev
p:3->4->5 p = p.next
第三层循环
rev:3->4->5 rev = p
rev:3->2->1->None rev.next = rev
p:4->5 p = p.next
第四层循环
rev:4->5 res = p
rev:4->3->2->1->None rev.next = rev
p:5 p = p.next
第五层循环
rev:5 rev = p
rev:5->4->3->2->1->None rev.next = rev
p:None p = p.next

end…
