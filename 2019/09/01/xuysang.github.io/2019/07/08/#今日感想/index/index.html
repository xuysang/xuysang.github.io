##今日感想（7.2）
今天公告挺少的，很快就完成了工作任务，10点之后就可以自己学习了。今天的感想会比较多，因为会涉及心里比较复杂的一面。
首先是学习python的函数的参数传递，刚好和周末看的一章有重叠，number,string,tuple是不能改变的，所以在函数外面的值，在函数内部被重新赋值后，依然函数外的变量是没有被改变的。相反，list,dict,set是可以被改变的，在函数中被重新赋值，函数外的变量实际值也跟着改变。

然后是照着书，实现了读取pdf内容的功能。之前一直不成功，今天就老老实实的照着书上代码敲了一遍，就实现了。具体的应用还没想，但是总觉得之后可以派上用场。

再接着是昨天实现了抓取鹏华基金指定日期的净值，今天想把抓到的数据，保存到我的数据库里面。从网上找了pymysql的教程，然后不断改格式，最后可以让数据入库了。其实还蛮开心的，明天想要实现，通过flask让数据库的数据可以在网页中展示出来。那肯定还涉及网页排版css、html5、javascript。总之至少实现一个过程。

然后就是目前的工作，和我对未来规划之间的思考了。我确实明白，鱼与熊掌不可兼得，但是工作我也是尽自己的一份心在做，所以难免还是会有所失落。但是确实付出的不够多，我不该妄想获得什么。如何调整自己的心态，如何认真做到无愧于心，然后把自己真正想学习的内容学好，这是我现在要关注和侧重的。情绪总会过去，下定决心的事情还是要一步步去实践的。
