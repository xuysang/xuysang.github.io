2019-7-17

说实话，有时候我不知道思绪多、感慨多的一天和水水过一天，哪种更值得。今天的感慨不少。1.我还不够，太不够了。2.不想让别人对自己失望，但是真的没办法。3.还是要有计划，踏实的走着，这是没有终点的，我永远以更好为目标。
今天用python实现单循环链表的创建，发现只有自己敲过代码，才会有更深的理解。实践，阅读缺一不可。比如类的定义，类的性质，实例化，具体应用。
今天不知怎么的，问候我的人很多，我才发现，只有把事情做成了，那才是说得出口的，刚刚才开始努力，不应当到处说。因为在被大家善意的询问时，先连自己的那一关都过不了。都说在达成目标的路上，过程中风景是美好的。我希望可以记录下来，也希望走到最后，我还保有头发。
做题目第一遍一定要写下自己详细的理解，不然后面第二遍第三遍看的时候，还要花很长的时间去想。
