宁静的夏天

六月和五月是不一样的心情。

五月是焦虑的，但充满希望，六月是看似平和的，但却多了一丝妥协，向现实低头的无奈。我突然能理解前两年的自己，每天按部就班的工作，是很省心的，没有太多焦虑感，也不用逼自己非得怎样怎样学习提升。回到公司工作一周，那种熟悉感马上又产生了，舒适区又开始向我袭来笼罩着我，面试学习新的内容好像有了挡箭牌，没有之前那么紧迫了。

但是我很不喜欢，鸡血凉了，之前的想法好似一个笑话。可是我不知道该怎么学习，在公司，即使给我一个下午的时间支配，我也不能有效的学习，很奇怪。明明觉得时间应当是宝贵的，可把大把的时间呈现在我眼前的时候，我又没那么珍惜了。

最近听到不少悲伤的消息，村里的某某或是认识的某某阿姨，出了意外走了。一直以来，我经历的离别，都是生老病死让人无可奈何却也顺其自然的，突然出现意外离世的，在我印象中不多，所以给我带来了一定程度的冲击。就像有句歌词说的，意外和明天不知道哪一个先来临，随着他们的离世，总觉得有某种联系断了。其实认识的人很多，但真正能在日后碰上面的，往往是少数。但不管能不能见面，知道他们在世界上活得好好的，会有一种安心感，突然走了的人，联系就断了，是真的真的，再也见不到了。

这一周我写了几个小程序，一个是抓取某些基金公司的净值，时间可以自己给定；一个是自动识别资本市场网的验证码，获取ISIN编码，虽然验证码有时候会出现错误，但是只要提高程序的执行次数，应该可以提高成功率；一个是自动下载巨潮资讯的基金公告。但在写这些小程序的过程中，我有这样的意识，就是自己的代码是不太严谨的，因为我只是存粹实现了我想要实现的功能，但是对可能会有的bug以及优化措施，我犯了懒，不太愿意去想。另外在写程序的时候也暴露了，我还是没有及时的将经验归纳整理，每次都要重新去翻自己以前写的，还是不够熟悉和顺手。另一方面就是，我能想到的可以实现自动化的还是太少了，应该多想多练。我真的很想实现自动发送邮件的功能，下周一定要投入研究。

换了一个靠窗的位置，外面的风景还不错吧。
