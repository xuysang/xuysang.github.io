刽子手游戏（uva489）
背景：计算机想一个单词让你猜，你每次猜一个字母，如果单词里有那个字母则会显示出来；最多只能错6次，猜一个猜过的字母也算错。

######题目大意：编写一个“裁判”程序，输入单词和玩家的猜测，判断玩家赢了（You win)输了（You lose)还是放弃了（You chickened) 每组数据包含3行，第一行是游戏编号（-1为输入结束标记），第2行是计算机想的单词，第三行是玩家的猜测。后两行只含小写字母。
样例输入：
1
cheese
chese
2
cheese
abcdefg
3
cheese
abcdefgij
-1
样例输出：
Round 1
You win.
Round 2
You chickened out.
Round 3
You lose

######解题的亮点：先写”伪代码”，然后转化成实际的代码。先写主程序，包括对函数的调用，再实现函数本身。在这个题目中，巧妙运用guess函数，将用户猜测的字母与原来的单词一一比对并记下猜错的次数，超过6次直接算输，同时记下猜对后剩余未猜的个数，为零是直接算赢。有个小技巧是猜过的字母再猜一次算错，解决的方案是将猜对的字符改成空格。

—###给出由O和X组成的串，统计得分。例如OOXXOXXOOO的得分为1+2+0+0+1+0+0+1+2+3.
代码

  #include<stdio.h>
#include<string.h>
#define maxn 100

int main()
{
    char s[maxn];
    int len,i;
    int t=1;
    while(scanf("%s",s)==1)
    {
    len=strlen(s);
    for(i=0;i<len-1;i++)
    {
        if(s[i]=='O')
        {
            printf("%d+",t++);
        }
        if(s[i]=='X') 
        {
            t=1;
            printf("0+");
        }

    }
    if(s[len-1]=='X') printf("0");
    if(s[len-1]=='O') printf("%d",t);
    }
    return 0;
 }

直接是答案的版本应该是：

#include<stdio.h>

int main()
{
    char s[maxn];
    int len,i,sum=0;

    while(scanf("%s",s)==1)
    {
        len=strlen(s);

        {

            }

                t=0;
            }
        }

输入方式

今天来总结一下c语言中输入方式的选择。
首先，最常用的是用scanf(“%d”,&x)的方式来接收输入的单个字符或是数字。此时，如果需要连续输入，可以使用for循环（倘若不知道要输入多少，可以在之前设置需要输入数的个数）。这里还可以采用while(scanf(“%d”,&x)==1)的方式来连续输入信息。在Windows下，输入完毕先按Enter键，再按Ctrl+Z键，最后再按Enter键即可结束输入。

getchar()可以ch=getchar()来输入字符，一个个输入，while((c=getchar())!=EOF)同上，需要按Ctrl+Z键结束输入。

同时，数组是很有用的，用来储存数据。首先需要定义数组s，在scanf(“%s”,s)中，它会读入一个不含空格、TAB和回车符的字符串存入字符数组s，s前没有“&“符号。它遇到空白字符会停下来。
