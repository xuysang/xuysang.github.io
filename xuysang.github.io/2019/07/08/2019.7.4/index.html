#2019.7.4
七月的第四天了。之前还信心满满的说从七月开始好好学习，到了第四天，那种无望的疲惫感又涌了上来，那种不知所措的感觉也跟着冒了出来。
是否我真的做什么事情，都无法真正坚持下去。看到真正的大神在自己面前，自己写的代码根本拿不出手，很丢人。想象我与真正专业人之间的差距，压力就大的像座山，根本不留缝隙就把我压得死死的。不知道为什么，计算机一直是我很畏惧的东西，所以我也很想真正去掌握它。但是一门专业的深度和厚度，真的不是我学几个月的皮毛就能达到的。我知道我不会放弃，但是接下来该怎么做，怎么做才能让自己有东西做，不要陷入无望的迷茫中，空怅惘。
