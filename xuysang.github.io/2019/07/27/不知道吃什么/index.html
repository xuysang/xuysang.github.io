不知道吃什么

一个月过的超级快，转眼又是一个周的周末。晚饭在炎炎夏日里，变成难以抉择要吃什么的难题，最后想到冰爽的可乐，点了金拱门。看到高昂的外卖费，又引发了我的焦虑。为什么吃个饭都这么不痛快呢~~懒洋洋的周末，本应该趁有空抓紧学习，学习，又被懒惰拖住了脚步，这样不可。最近发现了一个很好的python学习数据结构的视频，看完了单链表的那一块，真的觉得以前的我，压根是不懂。回忆着自己写了一遍，能测试成功还是很开心。我一定要把视频看完啊啊啊啊啊。
