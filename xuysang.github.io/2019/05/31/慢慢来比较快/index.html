慢慢来比较快

希望一会儿要写的内容，不要被未来疯狂打脸。
经过一两个月的焦虑，现在相对平和下来了，可能是因为想明白了一些事情。我之所以焦虑，是因为突然对某个东西有了欲望，而自己的能力却不足以支撑。这里有一个词要划重点，是“突然”。不由的又要扯到大环境，现在可能真的是一个“速度”的时代，网络的快速发展将人的耐心不断的打磨，什么都要求快。看视频二倍速，似乎可以节约时间看更多的东西。时间变得越来越珍贵，可是也变得越来越零散，因为“快”不允许在一件事上面耗费太多时间，所以做的事情是多了，但是很可能只是在做那件事的当下获得了满足，过后就遗忘。这也是为什么我回想工作以来这两年，却不知道自己做了什么，白茫茫。另一方面，网络的普及，让成功的例子随处可见，让我以为，真的可以随随便便成功，以为自己是幸运儿，以为不用费多大力气或者短时间苦一下马上能获得成功。概括就是一个字，“飘”。这真的是挺恐怖的一个字，与它对立的是我们中华民族传统品德的“脚踏实地”。
“突然”是一个短时间的概念，它让人的情绪在短时间达到一个高度，从而带来了刺激、振奋的感觉。并且为此，可能会做出很多冲动的事情。这不是说突然产生的念头，突然产生的冲动是不好的，只是告诉自己，突然的背后，要花更长更长的时间，来为这个念头、这个冲动背书。做好准备了吗？
