平凡的一天

因为下决心要早睡，10点半就睡，现在10点10分啦，连取个标题名字都好着急，想想今天，就很平凡的一天吧（其实每天都很平凡哈哈哈）。今天做了些什么呢，就继续学微电影搭建视频，我一定要早点学完还要把关键步骤也写出来，不然照着敲了一个，可别的收获什么都没有。然后呢就是找工作，流连各个找工作的app，一遍遍的刷着职位，看着达到那些条件还遥遥无期，投出去的简历石沉大海（苦啊）。然后数据结构学了栈，但是相关的leetcode还没有刷，看了python面向编程的两节课，不可思议，一天就这么过去了。效率真的是低下！今天天气很好呀，走出去买水果、买晚饭，挺舒服的，如果能吃上冰凉凉的雪糕啥的，就更完美啦。天天逛八组，我啊！今天刷八组看到了一个观点，觉得很好，这里记一下。就是一定要重视长时间的积累，提高效率，长期坚持，啊啊啊努力提高效率！
