list_2

学习链表，王峥老师说要熟练掌握五个常见的链表操作（单链表反转、链表中环的检测、两个有序的链表合并、删除链表倒数第n个结点、求链表中的中间结点，相对应的leetcode分别是206，141，21，19，876。反转链表已经在之前讲过，因此这篇主要讲的是剩下的四个。

在这四个操作里面，有三个可以通过快慢指针来解题。最开始，什么是快慢指针我也不了解，根据“五毒法”之第一步直接看答案解析，我知道了快慢指针的思想，人为的设置两个指针，一个走的快，一次走两个结点，一个慢，一次只走一个结点，说到这，求链表中的中间结点的思路已经出来了。

链表中环的检测，也可以利用快慢指针，因为倘若链表中存在环，那么这个遍历会一直持续下去，那么快慢指针终有相遇的时候。代码如下：

第三个是删除链表倒数第N个结点。这个一开始即使看解法，我也没有看懂（真的愚蠢）。其实从一开始我就没看清题目的意思，题中列表和N值是给定的。使用快指针明确走到第N个结点，然后慢指针开始走，当快指针走到链尾，慢指针所在的位置就是倒数第N个。当然遍历两边，直接找到倒数N的位置也是可以的

最后要讲一下合并两个有序列表，首先，是看有没有任一个表是空列表，那合并后就是非空那一列表。若两个链表都是非空链表，就分别拿出链表的第一个结点开始比大小，小的那个作为第一个结点，再把较大的那个与小的那个结点的下一个结点作比较，运用递归的方式，始终保持较小值的结点在前面

链表的五种操作方式终于写完了，其实除了快慢指针，还有其他方式也可以解题，这里着重介绍了快慢指针的用法。关于python的数据结构和算法，我觉得我对python面向对象这个特性以及用法什么的还很不清楚，然后我找到了两个不错的资源，希望能尽早看完，这里也贴一下。
