      栈与队列
    

  

      

    
    

      
        
栈和队列都是线性数据结构，并且栈和队列还蛮相似的，所以偷个懒，放在一起写。记得最开始学的时候，我也有疑惑，就是为什么要存在这种数据结构，这种不是相当于数组和链表的变形么，看了王争老师的解释：特定的数据结构是对特定场景的抽象。那么栈和队列的构造，就是为了一定程度上减轻使用数组和链表，因为前两者暴露了太多操作接口，使用时比较不可控，容易出错。
那什么是栈，什么是队列呢，不难理解，栈就是后进先出，一般都是想象一个凹槽，东西一个个放进去，最后放进去的是最先取出来的，大概是这么一个模型。队列就是先进先出，想象一个队伍，队尾不断有人进来，出去就是从队首先出去。所以最主要是要实现push和pop两个功能。
用数组和链表都可以实现栈。如果是用链表实现，压入和取出的操作都是在表头实现。






感觉用数组实现比较好理解。

2.验证回文串（leetcode 125）
给定一个字符串，验证它是否是回文串，只考虑字母和数字字符，可以忽略字母的大小写。
说明：本题中，我们将空字符串定义为有效的回文串。
示例 1:
输入: “A man, a plan, a canal: Panama”
输出: true


示例 2:
输入: “race a car”
输出: false


解法：
之前学习链表的时候，有一道判断回文的题目。当时给的用链表解题的思路是：用快慢指针定位到中点的位置，然后慢指针反转后面半个链表，再从头设一个指针，两个指针不断前进并且比较，如果一直一致就确定是回文字符串。（答案链接https://github.com/wangzheng0822/algo/blob/master/python/06_linkedlist/palindrome.py）
第二个思路，就是利用双端队列，直接pop比较两端的字符并且不断向中间靠近，当最后栈中的数据长短小于等于1时，表明是回文字符串。


第三个思路，直接利用列表的用法，和反转后的比较是否相等

push(x) – 将一个元素放入队列的尾部。

      微电影搭建二
    

  

      

    
    

      
        
这一篇主要介绍前台和后台，静态相关的内容。(page20)






前台后台的搭建。我个人的理解是，这里主要应用了前端的知识，通过html\css\javascript设计好了html,搭建了网页的样式，然后通过flask模块功能引用。因此知识点主要在怎么引入静态文件、定义路由、数据块等

首先是在templates的home和admin文件下面分别新建一个html格式文件(前台文件名为home.html)(这个文件是网页共有的部分，是作为一个基模板存在的，利用区块可在衍生模板覆盖)。


将前台网页共有的布局即页首、导航和页尾，放进html中，利用静态文件引入的方法，将相关的图片及代码放在static文件夹下，并用{{ url_for(“static”,filenaem=””) }}的方式显示，其中””里是图片及代码的路径。
接着是在导航与底部之间插入数据块。（p23){% block content %}{% endblock %}，用于衍生模板的覆盖。


接着是在templates中创建衍生模板（P21)


这里用到继承的概念，需要在templates中新建xxx.html，使用{% extends “home/home.html” %}，引入home.html的css和javascript文件，就是页首、导航和页尾。然后在数据块block content中，继续写对应模块的css和javascript(例如会员登录页面、注册页面、中心页面、列表页面、详情页面等等)。这里需要注意的是衍生模板中的链接，要使用{{ url_for() }},()中写的是定义路由时使用的端点名。
其中还有衍生模板中如果存在共同的，重复使用的代码片段(例如菜单栏)，那么可以单独写一个html文件，然后使用{% include ‘common.html’ %}引入需要的衍生模板中。

      难
    

  

      

    
    

      
        
想要做好一件事，不那么简单。从小听故事，很佩服那些能快速学习而且学得好的人，不由的也希望自己也能做到。但是好像总使不上力，就像跑步，老师们总在说爆发力，可是我始终也感知不到我的爆发力，我只觉得自己尽了力，但是这个度到底怎么衡量，我也不知道。似乎我做什么事情，激情都只能维持一小段时间，看到别人工作很激情，我也是羡慕的，可是回到自己身上，还是使不上劲。就像现在的学习编程，其实我要学的东西还有很多很多，我能做的也有很多很多，就是去学，可是我也会产生无力感，这已经是我最不抵触的事情了。当然，可能我最想做的事情就是躺在床上玩手机，就这么躺着，然后等良心过意不去了，爬起来学一会儿。


学习真的是急不来的。我很焦虑，想要快速学会技能好找到工作。可是，我不是天才，我每天学习的，也会忘记，需要再复习，还有许多未知的，等我去学习，静下心来，才能学得进去。


从小的学习，就好像是有什么已经在前面铺路，都是安排好的，照着做就好了，不用思考，不用计划。现在突然把这份权力赋予了自己，反而有些不知所措。不要求计划的多么完美缜密，但是心中有路了，总是好上道点。
